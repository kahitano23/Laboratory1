db.php

This file opens the MySQL connection.

Show code 
<?php
$host = "localhost";
$user = "root";
$password = "";
$database = "company_db";

$conn = mysqli_connect($host, $user, $password, $database);

if (!$conn) {
    die("Connection failed: " . mysqli_connect_error());
}
?>
Output:

No visible output. It only creates the database connection.

setup.php

This file creates the database and tables.

Show code 
<?php
$conn = mysqli_connect("localhost", "root", "");

mysqli_query($conn, "CREATE DATABASE IF NOT EXISTS company_db");
mysqli_select_db($conn, "company_db");

mysqli_query($conn," 
CREATE TABLE Employees(
emp_id INT PRIMARY KEY,
first_name VARCHAR(50),
last_name VARCHAR(50),
age INT,
salary DECIMAL(10,2),
city VARCHAR(50)
)");

mysqli_query($conn," 
CREATE TABLE Projects(
proj_id INT PRIMARY KEY,
emp_id INT,
proj_name VARCHAR(100),
budget DECIMAL(10,2),
hours INT
)");

echo "Tables Created!";
?>
Output:

Tables Created!

Questions 1-10

Each exercise now shows the SQL query first, then the PHP wrapper, then the expected output.

1. Display the complete name of all employees in uppercase.
Step 1: SQL Query
SELECT UPPER(CONCAT(first_name,' ',last_name)) AS fullname FROM Employees;
Step 2: PHP Wrapper
<?php
include "../db.php";

$sql = "SELECT UPPER(CONCAT(first_name,' ',last_name)) AS fullname FROM Employees";
$result = mysqli_query($conn,$sql);

while($row=mysqli_fetch_assoc($result)){
    echo $row['fullname']."<br>";
}
?>
Output:
ALICE SMITH
BOB JONES
CHARLIE BROWN
DIANA PRINCE
EVAN WRIGHT
2. Determine the youngest and oldest employee.
Step 1: SQL Query
SELECT MIN(age) AS youngest, MAX(age) AS oldest FROM Employees;
Step 2: PHP Wrapper
<?php
include "../db.php";

$sql = "SELECT MIN(age) AS youngest, MAX(age) AS oldest FROM Employees";
$result = mysqli_query($conn, $sql);

$row = mysqli_fetch_assoc($result);

echo "Oldest: " . $row['oldest'];
?>
Output:
Youngest: 22
Oldest: 45

10. Find the total project hours per city.
Step 1: SQL Query
SELECT e.city, SUM(p.hours) AS total_hours
FROM Employees e
INNER JOIN Projects p ON e.emp_id = p.emp_id
GROUP BY e.city;
Step 2: PHP Wrapper
<?php
include "../db.php";

$sql = "
SELECT e.city, SUM(p.hours) AS total_hours
FROM Employees e
INNER JOIN Projects p ON e.emp_id = p.emp_id
GROUP BY e.city
";

$result = mysqli_query($conn, $sql);

while ($row = mysqli_fetch_assoc($result)) {
    echo $row['city'] . " - " . $row['total_hours'] . "<br>";
}
?>
Output:
Manila - 270
Cebu - 200
